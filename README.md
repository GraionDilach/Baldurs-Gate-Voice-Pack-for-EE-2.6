# Baldur's Gate Voice Pack for EE 2.6+

## Overview
This mod adds the base voicesets of Baldur's Gate to the Infinity Engine (Baldur's Gate/Baldur's Gate II/Icewind Dale) Enhanced Edition games version 2.6 or above.

##Limitations
Due to inconsistency with other voicesets the Rare Action lines on the Beamdog sets are duplicated into rare selects as well.

## Version history
